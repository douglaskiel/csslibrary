animation-timing-function properties

  • Using 'ease' specifies an animation witha slow start, then fast, then a slow end. This is default.
  • The 'linear' property specifies a consistant speed throughout the animation.
  • 'Ease-in' gives it a slow start
  • 'Ease-out' has a slow end
  • 'Ease-in-out' gives a slow start and end
  • 'cubic-bezier(n,n,n,n)' lets you set your own values in a cubic-bezier function.
